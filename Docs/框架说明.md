# 游戏框架说明（初学者版）

> 适用：Godot 4.7 + C# + 2D 游戏

---

## 你有两个工具脚本

| 脚本 | 挂在哪 | 干什么 |
|------|--------|--------|
| `UIManager.cs` | Main.tscn 里的某个节点 | 统一打开、关闭 UI 面板 |
| `SceneLoader.cs` | Main.tscn 里的某个节点 | 切换游戏子场景（战斗、主菜单等） |

---

## 第一步：搭 Main.tscn

在 Godot 里创建一个 `Main.tscn`，节点树长这样：

```
Main  (Node 类型)
├── UIManager       ← 挂 UIManager.cs
├── SceneLoader     ← 挂 SceneLoader.cs
├── ConfigManager   ← 已有的配置管理
└── SceneContainer  ← Node 类型，子场景会加进来这里（不挂脚本）
```

然后在 **项目设置 → 主场景** 里把 Main.tscn 设为启动场景。

---

## 第二步：设置 SceneLoader

点击 SceneLoader 节点，在右侧 Inspector 面板里，
把 `Scene Container` 槽里拖入 SceneContainer 节点。

---

## 第三步：让游戏启动后加载第一个场景

在 Main.tscn 上挂一个简单脚本（或者直接在 SceneLoader._Ready 末尾加）：

```csharp
// Main.cs（挂在 Main 节点上）
public partial class Main : Node
{
	public override void _Ready()
	{
		// 游戏一启动，就加载主菜单场景
		SceneLoader.Instance.GoTo("res://Scene/MainMenu.tscn");
	}
}
```

---

## 怎么打开一个 UI 面板？

比如点击某个按钮后打开商店面板：

```csharp
// 在按钮的脚本里这样写
private void OnShopButtonPressed()
{
	UIManager.Instance.ShowPanel("res://UI/ShopPanel.tscn");
}
```

---

## 怎么关闭一个面板？

```csharp
// 在关闭按钮的脚本里
private void OnCloseButtonPressed()
{
	UIManager.Instance.HidePanel("res://UI/ShopPanel.tscn");
}
```

---

## 怎么切换场景？

```csharp
// 比如点击"开始战斗"按钮
private void OnStartBattlePressed()
{
	SceneLoader.Instance.GoTo("res://Scene/Battle.tscn");
}
```

## ⚠️ 已知限制和风险（给你说清楚）

### 1. UIManager 没有层级控制
所有面板都堆在一起，谁后打开谁就在最上面。
如果你发现弹窗被其他面板挡住，需要手动调整节点顺序，
或者在面板的 `.tscn` 里改 `CanvasLayer` 的 Layer 值。

### 2. 面板路径是字符串，写错不报错
`ShowPanel("res://UI/ShopPanel.tscn")` 这个路径如果打错了，
只会在运行时打印一行错误，不会编译报错。
建议在一个地方统一定义路径常量，避免写错：
```csharp
// 新建一个 UIPaths.cs
public static class UIPaths
{
	public const string Shop   = "res://UI/ShopPanel.tscn";
	public const string Battle = "res://UI/BattleHud.tscn";
}
// 用的时候：UIManager.Instance.ShowPanel(UIPaths.Shop);
```

### 3. SceneContainer 忘记拖就会报错
SceneLoader 的 `SceneContainer` 需要在编辑器里手动拖，
如果忘了，运行时会打印错误并什么都不加载。

### 4. 切换场景时旧场景被直接删除
`GoTo()` 会立刻 `QueueFree()` 掉旧场景，没有过渡动画。
如果以后想加"黑屏淡入淡出"，需要另外处理，但现在不用管这个。

---

## 总结：你只需要记住这两件事

1. **打开/关闭 UI** → `UIManager.Instance.ShowPanel(路径)` / `HidePanel(路径)`
2. **切换游戏场景** → `SceneLoader.Instance.GoTo(路径)`
